Por razones de seguridad y como buena práctica de REST, el protocolo HTTPS es obligatorio.

Ejemplo de código no compatible (OpenAPI 2)

Todos los ejemplos están en YAML

swagger: "2.0"
info:
  title: Swagger Petstore
  version: "1.0"
schemes:
  - http
paths: {}

Solución compatible (OpenAPI 2)

swagger: "2.0"
info:
  title: Swagger Petstore
  version: "1.0"
schemes:
  - https
paths: {}
