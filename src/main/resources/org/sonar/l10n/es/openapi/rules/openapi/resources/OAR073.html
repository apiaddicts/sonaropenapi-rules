La API debería incluir una respuesta 429 (demasiadas solicitudes) para indicar la limitación de la tasa

Ejemplo de código no compatible (OpenAPI 2)

Todos los ejemplos están en YAML

    swagger: '2.0'
    info:
      version: 1.0.0
      title: Non-compliant API
    paths:
      /items:
        get:
          summary: Get a list of items
          responses: 
            '200':
              description: A list of items
              schema:
                type: array
                items:
                  type: string

Solución compatible (OpenAPI 2)

    swagger: '2.0'
    info:
      version: 1.0.0
      title: Compliant API
    paths:
      /items:
        get:
          summary: Get a list of items
          responses:
            '200':
              description: A list of items
              schema:
                type: array
                items:
                  type: string
            '429':
              description: Too Many Requests
              schema:
                $ref: '#/definitions/error_response'
